# Facilities API Endpoints Documentation

This document outlines the FastAPI endpoints required for the Dashboard Facilities page.

## API Base URL
\`\`\`
http://localhost:8000
\`\`\`

## Required Endpoints

### 1. Get All Facilities
**Endpoint:** \`GET /api/facilities\`

### 2. Create Facility
**Endpoint:** \`POST /api/facilities\`

### 3. Update Facility
**Endpoint:** \`PUT /api/facilities/{id}\`

### 4. Delete Facilities (Bulk)
**Endpoint:** \`DELETE /api/facilities/bulk-delete\`

### 5. Bulk Import Facilities
**Endpoint:** \`POST /api/facilities/bulk-import\`

### 6. Log Facility Action
**Endpoint:** \`POST /api/facility-logs\`

See full documentation in the file for detailed request/response examples.
